HowTo Tag a Web Resource

  1. Create a javascript browser bookmark, known as a bookmarklet, with a name such as "TagoBookmark" and with this javascript code:
    javascript:location.href='http://www.topicquests.org:8000/#/tago/bm?type=b&url='+
            encodeURIComponent(location.href)+'&title='+encodeURIComponent(document.title)
  2. Log into the website
  3. Visit any webpage in the same browser as logged in
  4. Click the bookmarklet, which opens a form at the website, complete with the chosen site's title and URL
  5. Enter chosen tags
  6. Click Save
