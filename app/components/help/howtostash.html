HowTo Stash a Web Resource

  1. Create a javascript browser bookmark, known as a bookmarklet, with a name such as "TagoStash" and with this javascript code:
    javascript:location.href='http://www.topicquests.org:8000/#/tago/stash?type=s&url='+
            encodeURIComponent(location.href)+'&title='+encodeURIComponent(document.title)
  2. Log into the website
  3. Visit any webpage in the same browser as logged in
  4. Click the bookmarklet, which opens a form at the website, complete with the chosen site's title and URL
  5. Fill in the Notes form
  6. Click Save
